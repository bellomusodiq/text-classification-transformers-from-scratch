Text Classification with Transformers from Scratch
This repository contains a PyTorch implementation of a Transformer model built from scratch (without using Hugging Face transformers) for text classification. The goal of this project is to demonstrate the inner workings of the Transformer architecture, including multi-head self-attention, positional encoding, and encoder layers.

Features
Custom Transformer Encoder: Full implementation of Multi-Head Attention, Layer Normalization, and Feed-Forward networks.

Positional Encoding: Sinusoidal positional embeddings to retain word order.

Custom Tokenization: Simple character or word-level vocabulary building and padding pipeline.

Training & Evaluation Loops: Includes loss tracking, accuracy calculation, and validation splits.
